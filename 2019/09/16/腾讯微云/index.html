腾讯微云

发表于 2019-09-16 | | 阅读次数:

微云是新世代的下载方案，当然也有人用百度或是其他什么盘的，但在在我的多方体验下，微云是相对最优的。

理由是：

  • 一个月有5次免费离线下载的机会
  • 普通会员要10元（现在没有普通会员了，超级会员20一个月算是一般水平吧）
  • 相对不那么敏感（版权和那方面）
  • 稳定（和黑科云相比）
  • 快
  • 网页版就够用，不用下什么客户端（说你呢百度云）

虽然我吹了一波，但实际上也是矮子里拔高子，国内的互联网厂商没在你头上拉屎就是“良心”，给钱办事就算“良心”，什么鸟风气。

我还是自己建一个私有云好啦。
