如何减少无效面试

发表于 2019-11-08 | | 阅读次数:

求职者在面试之中实在太弱势了，招聘方几乎没有什么成本，有时甚至连简历也不看就叫人来面试，进门填一遍个人信息，看过之后说你不是我们需要的人才,不是你要的人才你为什么叫我过来啊，不看简历的吗，我TM坐车来你这面试，你花五分钟就打发我走了，我来来回回半天就耽误了，谁来补偿我啊。

简历的投递渠道

虽然市面上有很多的招聘网站，但我只推荐用前程无忧(51job)，我也试过其他的网站，但从来没有在上面获得过工作，而且自己编排简历很费劲。

51还有一个好处是筛掉了大量的“不正规”小作坊，这些小作坊恐怕连正经的HR都没有。51还有很好的筛选功能，发布时间，公司性质（公司自己填的真实性存疑），薪资待遇，福利等等。

真正的筛选

你在51上投递完简历，等到了面试邀请，这时才是到了真正筛选的时候，因为大量的公司jd很模糊，甚至就只有几句任职资格。
来电号码
看来到号码是否是公司电话，有时公司没有座机，办公电话也有是手机的，各种软件是能识别的，要是不能识别，要么是新公司，要么是垃圾公司（pass）。
确认招聘jd上的信息
大量的垃圾公司挂羊头卖狗肉，工作地点实际在某个偏僻县城啦，要长期出差啦等等，需要多加注意。
主动询问
大部分时候来电的是hr或者前台小妹，这时最好要求一次和直接用人部门的初步接洽，知道对方真正要什么样的人，知道对方是什么样的人，待接手的是什么样的项目。

最后

这样做了之后你发现几乎没有面试，很正常，现实就是这样的残酷，好公司不会老招人，流水线总是缺人。而且即使得到了有效面试也不一定能得到offer，很长一段时间里你都要忍受巨大的待业压力。

祝你好运！
