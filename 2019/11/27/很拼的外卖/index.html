很拼的外卖

发表于 2019-11-27 | | 阅读次数:
字数统计: 744字 | 阅读时长 ≈ 2分钟

天气转凉，虽说曾经无数次发誓不再用外卖，但冷的失眠，晚上睡不着，早上起不来，这几日都是11点多快12点才起来，纠结着起床后洗漱完毕食堂已经没有餐食了，加上小雨更不愿出门，所以还是用了外卖。

通常我点外卖只点麦当劳，麦当劳只点板烧鸡腿堡。原因一是板烧很健康——对！不是说快餐都是垃圾，鸡腿肉和面包哪里垃圾了？二是便宜，平台最近都是半价甚至于2.5折配合会员优惠，每单有时可以做到10元+5元两个鸡腿堡&顺丰配送，十分优惠，食堂里一块干巴巴的鸡排都要6元了呢！

言归正传，虽然也有很多外卖小哥|大叔|大姐送餐很随便，有时甚至有冲突，但还是有两个事挺让我感动的。

送上门的麦当劳

其实如果要求的话都是会送上门的，但这个小哥在我没要求的情况下就自己送到了门口，之前他也送了几次给我，按他的说法我是老熟人了，所以后来都是上门的，我其实是有点不适应被人知道自己的准确住所的，所以一般也不会让送上门，这小哥真是挺生性的，真的很好的服务，我这三楼，背个送餐箱上楼不会太容易，要是我的话还不如让客人下来，自己刷刷手机缓一缓，这么认真，还真是挺感动的。

雨天送上门的披萨

我本来打算点尊宝披萨的，在平台上下单后发现自己点的不是“尊宝”，而是“尊宝宝”虽然商家简介称自己是子品牌，但我在网上也没有搜到可以应证的东西，大概就是个山寨吧，logo做的还挺像回事。

外卖点餐就是不能漏过细节，不然就很可能会达不到预期，就浪费了，我要的是“尊宝”，而不是“尊宝宝”，反正还刚刚点，就取消吧。这样我点了取消，很快对方打了电话过来说已经开始做了，不能取消，要是取消了就要自己赔了，我不好直说你是山寨，就说这配送太慢了，对方给我一个保证——15分钟内送达，我不好拒绝，就同意了，天下着雨呢，交通又不好，怎么可能呢？

然而真的送达了，还比预期的快了2分钟，配送阿姨——我也没仔细看说不定只比我大一点，但生活艰辛，配送阿姨说自己是闯红灯过来的，真是拼啊。真不容易，也只是50元不到的东西，刨开配送费也就是30多，却可以让人做到这样的事，真不可思议。
