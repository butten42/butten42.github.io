搬家

发表于 2019-11-25 | | 阅读次数:
字数统计: 717字 | 阅读时长 ≈ 2分钟

搬家是最让人头疼的事之一了。

虽然我的思维模式工具箱里老早就配了一把奥卡姆剃刀，但真正用的时候并不多。所以我搬离的时候东西一定是大于我迁入的时候。

迁入时候拿的一定是必须品，而生活一段时间后总是会被消费主义所诱惑，买了大量自己不需要的东西，又不舍得丢掉，越积越多。当然也不都是无用的，春天不必准备冬天的衣服，中途买也是可以的，加上时间会改变你的想法，以前觉得不重要的现在变成必需品，以前没有发现的东西变的令人愉悦，生活总是会有意外和惊喜，人不能谋划。

但是搬家的时候就要做一些取舍了，这时就是“水落石出”的时候了，什么是真正的垃圾，什么是自己想带到下一个居住点的。

书

我很早就不在出租屋里买实体书——除非忍不住，除了技术类学习类的要反复翻阅的，我几乎不会再去买实体书了。再说现在的娱乐方式也多的不得了，买了也不会看。

家具

“房子是别人的，生活是自己的”听起来很有道理但事实上，租别人的房子，你的生活也是别人生活的一部分，就事论事，不要误以为出租房是自己家，出租房就是没有家的体验，这是个冰冷的事实，越早认清越好。

我曾经在出租房里买过床垫，书桌，椅子，冰箱……最后都不得不额外的花时间来处理掉，我的目的是住到自己的家里。而这是别人的生意场罢了。

当然也有不得不的时候，我一直后悔没有买个新床，出租房的床实在太操蛋了，大大的影响了我的睡眠，睡眠有问题白天就没啥精神，而且一个床也不贵。

找快递

要是是换城市的话，就要找快递公司了，我十分鄙视明确的廉价快递，十分没谱，收费也不便宜，也许以后要便宜我直接邮政包裹了，要速度直接京东或是顺丰了。

找搬家公司

货拉拉很棒，东西少的话直接叫个滴滴也成。

和房东交接

个人房东一般比较好说话，就是公寓什么的就很麻烦。也会有大量的莫名其妙的扣费点，没地方也没时间精力和他耗。

离开

你离开了这个曾经生活过的地方，再也不会回来了，你的生活和荷叶上的水滴一样没有留下如何痕迹。
