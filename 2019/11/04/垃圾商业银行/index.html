垃圾商业银行

发表于 2019-11-04 | | 阅读次数:

总结下我经历过的垃圾银行，因为这些都是商业银行所以有这样的标题。

浦发银行

开了一张花悦卡，说是有送花，但根本没有，说是送话费，但其实是话费满减券，要不就是要满足这样那样的条件才行，根本是公开的虚假宣传。

到网点开卡，居然没有大堂经理，而且柜台只有两个，其中只有一个在办业务。我先是被叫到自助设备上开卡，之前我了解到这个是要到柜台的，但压根没人理我，我试了一下是无法自助的，其实只要在柜台上做一个远程身份核实，但柜员弄半小时，而且我已经和她说了我试过了自助设备，她还要去一趟，最后一个年纪稍长些的人提醒后才放弃了，反反复复也不知道在干什么，中途就光给我推销信用卡了，最后和我说没这么快，留下我电话，说什么好了通知我，你先回吧！

你猜我还会不会去浦发银行。

中信银行

网上申请了微信联名的微加卡，已经发了短信要我去网点面签，等我去了被引导到柜台，柜员说没有卡片需要激活，说可能短信发的太早，让我回去。这是第一次。

大概过了一个礼拜，我接到通知要面签后才发实体卡，于是在礼拜天去了网点，因为去之前查询了营业状态，的确是营业的，于是我去了，结果以没有现金柜台值班为由又让我回去。这是第二次。

第二天我又去了，很快大堂把我引导到了自助设备上，等等你有自助设备怎么第一次不让我用？白跑这几趟，最气人的是居然面签失败，审核成功了面签居然失败，这下卡也不用寄了。

客服很难打通。

你猜我还会不会去中信银行。

华夏银行

柜台太慢，而且傲慢，明明柜台能做的事非要偷懒推给客户做。

你看我还会不会去华夏银行。

招商银行

一开始我对招行还是很有好感的，直到自己办卡的时候发现他们也是各种小心眼，办完卡之后垃圾信息都来了。

你说我还去不去招商银行。

其他

交通银行虽然也有垃圾推销的，但用卡的时候还是比较舒服的，有真正的实惠。
农行很垃圾，但我实际上没怎么用农行所以不好说。
建设银行的柜台是我第一次见到爆粗口的，对，其实我也不会去建行。
最舒服的还是中国银行，嗯，中行赛高。
