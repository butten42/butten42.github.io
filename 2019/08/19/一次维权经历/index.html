一次维权经历

发表于 2019-08-19 | | 阅读次数:

缘起

在18年初买的戴尔XPS 9560在使用5个月不到有一天接了外接电源正常使用忽然无法识别电池，不能充电了，我首先到了离我最近的深圳某家官方售后，得知需要更换主板价格为1399元（美版戴尔国内不保修），当时我觉得太贵就没有换。

然后去了华强北，电脑城的奸商和我说当天就能拿机子花费在200~300内（当然是经过讨价还价），我去了两家但没有一家像他们所承诺的那样可以当天拿机器，而把机子放到奸商那里过夜还是蛮担心的。毕竟买回来半年不到。

最后就到了今次了，7月中旬，在机器使用了一年半，发生故障后一年，我的主板电池没电了，每次断电后启动需要重新设置时间，也不能带出去使用，开始觉得有些麻烦，加上自己想要一段时间的远离电脑的日子就在闲鱼上找了一个当地的维修店，聊了聊，觉得还算靠谱就拿去那里修了。

然而却是噩梦的开始。

过程

内容：主板维修

价钱：我们一开始在闲鱼上说的是一百多，但他看到了机器后改口要400多，被我反复还价到280。

维修时间：一开始说一两个小时就可以，后来修不好就说要个两三天，等过了两三天再问就说等物料，还要个一两天，又等了个一两天又说没有用还是要等，到了第十天的时候我要求他退回给我，他口头答应，但等到我去他店里他却已经在他给我的上班时间里提前下班了，第二天我提前半个多小时通知他来取机，等到他店里的时候他才不太情愿的开始把我的电脑装回去。整个过程里都是我打电话过去问，对方承诺的通知根本没有，完全不把我当回事，而且取机当天还频频辱骂威胁打人。

在拿到机子的当天晚上，也就过了几个小时，电脑就断电不能开机了。交涉无果，发起维权。

维权过程

途径：12315电话，12315网站，税务稽查举报，食药监举报

电话还是很快就能接通的，案情描述完了后分配处理单位，第二天就有人和我聊这事了，专员听起来是个60到70年代出生的男人，第一句话就和我说，这个比较麻烦，没人懂这个，需要我提供直接证据。

我提供了他在论坛里的帖子，证明我的机器之前完全正常，而现在却有断电不能开机的故障，是对方的责任，没有得到支持，要我去官方售后做检测报告，需要有直接证明，但官方给出的报告仅仅是对送机的状态描述，没法直接证明。最后专员给我撤诉了。

PS：我还举报了他在店里卖蜂蜜，属于超范围经营。给出了他店内有两罐蜂蜜的图片和闲鱼上的交易记录。

专员认为没有贴上价签不能认定为商家交易，维修店也辩称是和朋友间的转让，闲鱼也只是个人，不是商业经营。

最后我指出他的经营信息没有及时更新，按照公司登机法是要处以一万以上的罚款的，专员才回复会责令变更。

网站基本没用，我在网上的投诉分配的专员给我的电话没别的就是和我说不知道怎么会分给他了，这不归他管，让我找另外一个分局的，但是网上的处理单位里又没有那一个分局可选，他称自己也没法转让案件到那边。

税务那边我提供了对方在闲鱼上的交易截图，认为没有上税，税务局认为属于个人二手交易免税。

食药监和12315合并。

总结

投诉立案到撤诉大概一周时间吧，心力交瘁。

工商对投诉类案件仅能协调，没有执法权。

公权力是很惰性的，所有举证都要由主张者自行提供，而就算得到了支持，就算是得到了赔偿，也仅仅是得到了当时被侵害的赔偿，大多数时候是为时已晚，而且中间的维权花费过于巨大，而侵害方，仅仅是把侵害的吐出来而已，这还是最好的情况下。

电脑城是不能去的，其实只要我在去之前查过了地点的评论就会发现那个地方全是骗子，不过可能没什么用吧。
