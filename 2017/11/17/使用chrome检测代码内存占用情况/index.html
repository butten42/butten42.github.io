使用chrome检测代码内存占用情况

Veröffentlicht am 2017-11-17

如何使用chrome开发者工具查看程序内存占用情况

工具

Chrome javascript profiler

开始

打开Chrome javascript profiler
对你想要分析的时段进行录制
默认录制完毕会显示heavy（重度内存占用）的列表
进行降序排序后找到内存占用最大的既可。

例子


在这里用我的打飞机游戏为例，我录制之后，出现这份图表：

  • Self Time是函数自生调用花费的时间
  • Total Time 是函数在录制期间一共花费的时间
  • 后面淡灰色的百分数为内存占用情况
  • Function 里显示具体的函数或者模块名称，点击左边的倒三角可以显示出更加详细的调用关系，比如我这有好几个函数调用了drawImage这个函数方法，分别是Bullet.draw,Background.draw,Plane.draw,Bao.draw,goToCover。录制期间一共占用了11..94%的内存
