  • js指南

  • 高级教程

快速入门

起步
  • 介绍了JavaScript 的基础功能，比如变量、字符串、数字、数组等。
  • 介绍了常见类型的代码块，比如条件语句、循环、函数、以及事件等。
  • 介绍对象的设计思想和语法、如何创建对象，并解释 JSON 数据是什么、如何使用。
  • 了解有哪些 API，并学习使用开发过程中最常见的 API。

    有意思的是其中一个例子‘制作一个随机数的猜测游戏’，技术上很简单啦，无法是使用random生成随机数，然后提示用户的数字大小，再处理是否成功猜测的事件。在这里教程很详细的分化了每个步骤，和FCC上的user story一样，都是要求你“像一个程序员一样思考”。

    回顾了一下字符串的方法：slice() toLowerCase() toUpperCase() replace() split() join() toString shift unshift
