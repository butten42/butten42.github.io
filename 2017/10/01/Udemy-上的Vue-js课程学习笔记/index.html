Udemy 上的Vue.js课程学习笔记

发表于 2017-10-01 | | 阅读次数:

缘起

之前学Vue的时候按照尤大的建议把官方文档对应内容过了一遍，大概知道了这些东西，于是就打算做个项目练手，我想todo-list应该是个比较合适的对象。于是就找来看过了源码，发现自己还是不怎么样，很多地方还是不清楚。
于是就在网上找视频看，在油管上有个排名靠前的教程讲的挺好的，就拿来做教程了，但是他只有三段试看视频，在信息页里我找到了它在Udemy的课程链接，只要10刀，于是就买了。

简单介绍一下课程吧

课程的授课形式和udacity等线上课程一样，都是视频一些文章和练习（在视频内非及时的）相关链接等，并有专门的问答区，不过他的教学视窗默认就全浏览器了，这样使得我一边看视频一边改代码不可能了，有点不太习惯。视频共有15.5个小时，分为14节（其中的第一二节正是在油管上的试看视频）：

  • Getting Started
  • Fundamentals of Vue.js
  • Exmaple application #1: E-commerce(10)
  • Deep dive: The Vue instance(14)
  • Setting up a Development Environment with Webpack & Vue CLI(7)
  • Components(19)
  • Example application#2: Mail application(18)
  • Mixins & Filters(5)
  • Forms(9)
  • Animations & Transitions(16)
  • Routing(SPA)(31)
  • Connecting to Servers through HTTP(12)
  • Vuex(36)
  • Conclusion(2)

课时最多的是Routing（31课）和Vuex（36课），想必是Vue学习的难点吧。
