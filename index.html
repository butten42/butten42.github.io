A Polite Request

mayuko

发表于 2018-10-22 | | 阅读次数:

由于找工作上的失败心情低落，在家待业了两个礼拜，日夜颠倒，日常就是打打游戏混日子，吃个饭居然要一个小时，每天和自己说这样是不行的，要做运动要早睡早起要读这个那个要投简历，给自己设定了一个又一个闹钟，但最后都没有用，我沉浸于这样一个惨淡的负面情绪中，觉得什么都不会好了，什么都不会改变。

但是我心里还是知道这样会把我引到一个什么样的地方，而那并不是我所希望的未来。于是在上个礼拜天我和自己说一定要洗个澡，要看一些招聘投一些简历，明天准备一下面试。

然而事实上我仍然是打了一天游戏，临到夜里10点的时候我手机里的闹铃响了（这是我提醒自己早睡的铃），我在游戏里也受到了挫折就关掉了游戏，但并没有让我去上床休息的念头产生，我又刷了1个半小时的垃圾视频（我十分了解那就是垃圾），洗漱完毕已经是凌晨0点了。这时我刷到了一个youtuber的视频，视频的主人叫Mayuko。

Mayuko是在硅谷的日裔程序员，她年龄和我相仿，行业和我相仿，拥有和我一样的烦恼，在6分钟的视频里提出的观点我基本是赞同的。对于高薪稳定的生活的个人的生命意义的探索之间的选择，最终起决定因素的不是选择能带给我的东西，而是对于失去的恐惧与担心。

在下决定之前的反复自我询问使得我们更加了解自己，知道选择将带来的是什么。对个体意欲的表达大多数情况下是有很大的风险的，但在可能的状态下我还是会尽量做这样的选择，当然稳定当然很重要，自由并不容易。

the right thing

发表于 2018-04-15 | | 阅读次数:

结束了漫长的赶工发包的一周，离开公司的时候已经是晚上十点一刻之后了，搭地铁回到家里应该就11点之后了。道友和我是最后离开的，下电梯的时候道友说我应该多看点自己工作上有用的东西，言下之意是我平时看那么多python的东西又不能对我的职业生涯有帮助（可能也暗示我前端代码写的不怎么样，可是我认为他平时也不怎么review，只是形成了固有的印象，没有看到我在较新的项目里已经写了很多优秀的东西）。

无论如何，要是我一心要做好前端开发，那么我是应该去看些js方面的东西，多做点前端的项目。python又没有什么帮助，优先级是不如js高的。

可是我没有这么做。

我每天都在写自己不想写的东西（公司的项目就是一坨屎），为什么下了班不能做点我想做的东西。再说我一开始就是学Python的，只是学前端只是一时之计，我当真要把它做一辈子吗？未必吧。

做前端这段日子里我是体会到了什么是“前端娱乐圈”，都是靠吹啊，单纯的技术并不好使，我再也不相信什么“Make the world a better place”了。程序员们自己身上还一坨屎，还要去搭救别人，老老实实搬砖好了。

我还能做多久的前端？甚至我还愿意呆在技术圈多久？现在我还对Python有兴趣，可是我不知道能持续多久，要是我再也不能在写代码上找到任何创作乐趣，那工作简直是suffer。而且薪资又不高，还要在人间地狱的大城市里做一只磷虾，空耗我余额不足的青春。真还不如在银行的日子，tm的真的做了个愚蠢的决定。

那么我现在该做什么才能算的上是”The right thing”?

离开银行的时候我是有计划去意大利留学的，但出来之后我发现自己没有这个魄力把所有的钱都直接拿去留学。而且，别人留学是很早就在做准备了，我的这个东西太迟了。

而到了今天我对新的东西已经不那么敏感了，也没那么想去意大利或是别的地方，太阳底下没有新鲜事，磷虾都是差不多的，出去一趟的收益似乎不太能弥补成本了。

我想做技术总是不会有错的，这个东西在哪里总是能有口饭吃的，但事实上圈里的情况太乱了，这个混乱的情况可能在某些人眼里是机会，可是我作为一个新手还是需要一些规则的依靠。

单纯的技术是不可行的。

我们作为一只只磷虾，拥有的最宝贵的资源只是时间而已。要获得真正的核心竞争力只有靠不断的砸时间积累的东西。

而我几乎是每过一段时间所有的东西都被清空重来了，到了今天手机里会打电话的朋友都屈指可数，也没有什么硬技术（我现在的开发水平也只是比菜鸟好一些），换言之手里的牌太小了，靠这些牌想要打出我之前在银行的状态甚至都十分吃力了。

麦里浩径一日行

发表于 2018-04-06 | | 阅读次数:

清明是想约人去徒步香港，但始终没有约到，本来是要取消这次出行的，但临到出行的前一夜我忽然意识到自己这一年都没有出去玩过，所以最后虽然没有人陪我还是一个人去了香港。

攻略篇

出发前的准备：通行证，八达通，香港流量包，遮阳帽，kindle，一瓶水，一部iphone 5s 一部索尼z3c

主要任务：徒步麦里浩径第二段
次要任务：帮人带货

关于麦里浩径

麦里浩径是香港一条著名的徒步路径。全长100公里。以西贡北潭涌为起点，绕过万宜水库，由东向西横贯新界，以屯门为终点，共分为十段。麦里浩径第一段以北潭涌为起点，到浪茄为终点。沿途主要绕着万宜水库兜大半圈，全程约10公里。该段前部分以公路路面为主，后半段开始走山路。麦里浩径第二段以浪茄为起点，途径西湾山、西湾、咸田湾等地，终点在北潭凹，全程约14公里。该段以山路和沙滩为主，沿途经过香港最美的海滩——大浪湾。

故事篇

由于是临时起意的，所以起的比较晚，好在已经做好了功课，去香港也不用太多的准备，所以出发并没有太仓促，不过节假日出游肯定是人山人海的，虽然在过关的时候一度想返回，也只能说服自己这就是自己想要的。

千辛万苦过了关搭乘地铁到了沙田，沙田巴士总站就在地铁出口下一层，并不难找。
在巴士站里找到1站台，299x的巴士前面已经排了好长的队伍了，我刚刚“哇”了一声，在我后头的本地人就说怎么这么多人，我前面的师奶很鄙夷的对本地人说“旅行团咯，痴线，坐巴士”，还好她应该没有把我归到“旅行团”里去。
在师奶前面有一对结伴而行的女生，我后来又遇到了她们。

公共巴士都是双层的，我当然是去楼上找位置了，不过也不是第一次来香港了，深圳也不少高楼大厦，所以也没怎么看城市，反而是到了郊区的时候发现别有景色。

在西贡下车后我觉得单单靠一瓶550ml的水应该是不能支持全程的徒步的，就去711又买了1.5l的水，11港元，还是挺便宜的。

接下来我要去村巴搭乘点，就在附近的麦当劳外面，正好可以去吃个午饭。

不多时到达搭乘点，已经有不少人在哪里等候了，据说是等了好久都没有车，而在公告牌上显示的是到3点才会有车，我就先去了麦当劳。等我出来的时候队伍已经不见了，前后也就十几？二十分钟。在公告牌上是有电话的，但是我只买了流量包，不好打国际电话。在这个时候我发现前面等车的两个女孩也来了，相互攀谈了一会之后知道目的地一致，她联系了公告牌上的人。那边通知2点会有一班车。还真是挺农村客运的。我提议要是不来的话我们几个人可以打的士一起去，虽然会很贵。附近有个英国人和她的女伴后来我们也反复遇见了。

好在2点的时候有一辆加长白色面包车开了过来，司机在候车点附近停了下来，打开了车门示意我们上车，村巴也是可以刷八达通的。价格稍贵，那两个女孩似乎是四川重庆的，全程方言不停，似乎并不想和其他人分享旅程，我就全程没有再和她们有交谈。景色还是不错的。

到达西湾亭的时候我才知道这个地方真的就是一个亭子。我稍微休整一下就上路了。临行之前遇到一个香港本地大叔，也想去徒步，然后问我路程和巴士番号，我不会主动讲粤语，但他用粤语问了，我也就用粤语回了，到最后他才发现我不是香港人。

出发的时候香港大叔估计去上卫生间了，重庆女孩已经走了，英国人正好和我一起出发，但是他们有自己的团体，我并不想没由来的插入，而且英国人太吵了，我很快超过了他们。

虽然我之前看地图是走这条路没有错，但是路上遇到的都是迎面而来的人，心里有些慌，还好很快遇到了重庆人，两下一合计就停下来问人，并没有错，我们就又各自出发了，之后虽然又遇到一次但完全没有交谈。

走了一段路不知怎么的前面的洋人就开始全部脱衣服了，光着膀子像是要干仗的样子。

原来前面就是海滩，而之前遇到的英国人也开始在海滩上搭帐篷了，事实上路上几乎所有的洋人目的地都是海滩，简直不知道海滩为什么对洋人这么有吸引力。

连洋人大妈都兴奋的用中文和我说“你看，这个船”，我怎么就总是觉得这么boring呢？

做什么事都没有精神，果然是我浪费了我的黄金时代的惩罚。

身上开始散发出一股的衰败的气味，做什么都是delay了，

很不开心，不知不觉中身边已经没有人了，即使后来又遇到了人，也不再能那么轻松的接受了。

一股未来会越来越坏的感觉涌上心来。

我要是回去写代码这个假期说不定就能把python的博客后台搭起来了，然后不就全栈了，不就加薪了，不就能有更多的机会了吗，为什么要一个人来徒步呢，什么问题也解决不了不是吗？

我穿过一群又一群人，做一个纯粹的旁观者，从他们的话语里猜测他们的身份还原他们目前的状态，但并不参与，也并不破坏他们的状态。成年后开始慢慢明白自己所做的事都会积业，所以还是小心行事，变得越来越保守，boring。

太阳底下无新鲜事，这个世界的本质就是boring的吧。

路过赤径，这里有个国际青年旅社，要是徒步的话这里是个留宿的好地方，可惜我应该是不会去了。

行程将尽我踏上返程，巴士在路上被一群牛堵住了去路，我想起在吉安的日子，那时候怎么会没想到有一日自己只要想的话可以随时来香港玩，早上出门，晚上还能睡到家里的床上，可是香港也不过如此。除了一些政策上的优势之外，香港在我看来极不宜居（深圳也差不多），我都找不到还要来香港的理由。

黄子华都说了，香港是“李家城”，资本家的地头而已，哪来那么多人文情怀。

下载已死

迅雷居然把主动屏蔽了盗版资源，本来就越来越难用的软件现在更加鸡肋，然而回想这两年自己下载的东西是越来越少了，一方面是没有时间了，另一方面是b站上的在线视频资源也丰富了，网速也快，在线看也不卡了，很多节目也都是一次性的，没有下载的必要。

我新买了台电脑也就是下载了游戏，搬运了之前的电影和图片资源。然后日常使用浏览器自带的下载就够了，想下载一个大一些的片子的时候是会有些困扰。

我本来整了个aria下载器，一个非常程序员的下载工具，下载也还好，就是重启路由之后之前的任务不能保存，然后偶尔还是会有一些bug，用户体验是不太能和市场上的商业软件相比的，也许有时间写写这个玩意的使用和配置。

然后又在知乎上搜索迅雷的替代方案，试了一下那些东西，并么有更好。

反正也没有想看的电影想追的剧，闲暇时间打游戏，也没什么机会和龟毛下载器打交道了，没有好用的就算了，我也没有时间在这上面耗了。

回乡记

发表于 2018-02-14 | | 阅读次数:

2月11日从广西北海返程，先飞南昌，飞机也就一个半钟头不到，但在南昌坐机场大巴到南昌站也花了一个半钟头至少。在车站等了一个小时终于上车，这时已经是19：50了，到达吉安的时间至少是22：00，这时已经没有直接回家的公交了，我叫了个滴滴，在拥挤的火车站广场我们互相找了半天终于见了面，当然也是由于我的主力手机没电了，只能用备用机临时叫车然后发现没法打对方的电话，之前因为发现不认证就没有办法叫顺风车，所以我换了已经注册认证过的手机号，但在这时发现滴滴给拨出去的号码还是用的我换手机之前的号码，这样大家都没办法联系上对方，这是我对滴滴最失望的一次。

23点过10分的样子我回了家。家里小区的门禁我刷了好久才发现要手动拨开才有用，骂了声‘SB’，上了破破烂烂的电梯回了家发现我房间的钥匙也不见了，这时我妈才出来告诉我她弄掉了那个钥匙，我洗了个澡，把keep上的几个简单的任务完成了。继续看完了“神盾局特工”的第十集。把手柄配置好玩了会巫师，不久就由于过热保护自动关机了，SB散热设计。

家里的床睡的甚至不如深圳的出租屋的舒服，失眠对我的脾气和记忆都造成了巨大的损伤。

  • 第一天，2.12，头天晚上大概2点睡，早上10点起，在市区买了裤子，本来想再买一件现在穿的衣服，但是没有合适的，买了水果和药品，就回家了，这天的其他事我记不清楚了，我吃过了午饭后在房间里躺着休息了两个小时，之后似乎我看了敦刻尔克，吃了点零食，刷了下京东，给电脑换了启动配置。
  • 第二天，2.13，1点睡，中午12点起，家里的SB被子太轻了，我冷的瑟瑟发抖，必须要等到热量重新回到我身上的时候才能起床，在自己家里都过的和一个流浪汉一样的生活我这一天都没劲，吃了饭后（从这天之后我就只有早晚两餐吃了）我本来想去干掉我keep上的配眼镜任务的，但是等我走到那里的时候发现自己已经没有力量了，我不想在这种状态下和人会面，我中途去看了老家，一片废墟。回家整理了一下房间，晚上玩了巫师。
  • 第三天，2.14，2点睡，中午12点起，我根本没有盖被子，巫师玩完之后我就和衣而眠了，当然不是什么好的体验啦，所以也起不来。我又去了一趟市区，把衣服买了，在回程去看了眼镜店，店里的配镜师傅十分和蔼，给了我十分多的建议，甚至跟我说要是觉得配的眼镜对于我的生活可能没有太大的帮助可以选择不配，这个检测都是免费的。简直赞，然后我也不知道怎么就没有坚持完成我的任务了。
  • 第四天，2.15，1点半睡，中午12点起，重新铺了床，稍微好一些了，但是也许还是太累了起来的时候已经快十二点了，我在床上刷b站知乎，其实都是所谓的“一时之乐”，就是不想起床，在2点之后实在不行了就起来洗漱吃饭了。

本来有想去找朋友聊天的，但发现大家早就分开了，再也聚不起来了，没必要再勉强了，我还是一个人吧。

噩梦惊醒，他长舒一口气，已经两年了，他还是没法忘记那一天。

那是圣诞节前的最后一个礼拜天，他站在教堂门口，试图思考一些东西，然而却只有一片空虚和不确实感，最终他离开了这里。

他看见了她，饱受沧桑的样子，和他记忆里的已经不太像了，但是他仍然记得她，认出了她，可是在弥撒的最后一节结束之前他悄悄的离开了，在教堂里他没有选择去打扰她，他不知道为什么，觉得这就是最好的结局了。

可这真的是最好的结局吗？

这之后他常常会想起她，他不知道为什么会变成这样，就像她在最后的日子里质问他时一样。

Merry Christmas.
